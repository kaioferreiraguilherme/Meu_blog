# Meu Blog — Marketing Digital

Site simples de blog criado para GitHub Pages.

## Estrutura
- `index.html` — página inicial
- `style.css` — estilos
- `script.js` — lógica (lista de posts, busca, menu)
- `posts/` — cada arquivo HTML é um post (post1.html, post2.html, post3.html)
- `assets/` — imagens e favicon

## Como publicar no GitHub Pages
1. Suba todos os arquivos para o branch `main` (ou `gh-pages`) do repositório.
2. Vá em **Settings → Pages**.
3. Em **Source**, escolha `main` branch e root, salve.
4. Aguarde alguns minutos e acesse `https://<seu-usuario>.github.io/<nome-do-repositorio>/`

## Editando posts
- Edite `posts/post1.html` (e post2/post3) para alterar título, imagem e conteúdo.
- Se adicionar imagens, suba para `assets/` e atualize os `src`.

## Contato
Coloque seu email no `contato.html` (action mailto) ou configure Formspree para formulário funcional.
